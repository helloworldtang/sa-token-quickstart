# Sa-Token 基础示例

本模块包含 Sa-Token 的基础使用示例，是原单模块项目的代码。

## 快速开始

### 启动应用
```bash
mvn spring-boot:run
```

### 访问文档
启动后访问 Knife4j 文档：
```
http://localhost:8080/doc.html
```
